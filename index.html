trackhub by medecau

A Bittorrent tracker hub.

trackhub maintains a list of public trackers and acts has a tracker but redirects all requests to the trackers available in it's list. The listed trackers are regularly checked and redirects are only made to responding trackers. A running instance is available at

Dependencies

trackhub was designed to run on Google App Engine, any other environment may require code rewrite.
